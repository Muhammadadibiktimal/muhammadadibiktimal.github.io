About Me

Halo, izinkan saya untuk memperkenalkan siapa saya. Nama saya Muhammad Adib Iktimal. Orang biasa memanggil saya Adib. Saya anak kedua dalam keluarga saya. Saya lahir di Jakarta, 18 Januari 2003. Saat ini, saya tinggal di Gunung Sindur, Bogor. Saya sangat suka musik Selain itu, saya juga tertarik dengan Desain Grafis. Saat ini, saya memiliki beberapa portofolio Desain Grafis profesional untuk media sosial.

Saya lulusan SMK AL-AMANAH tahun 2021 disana saya mengambil jurusan RPL (rekayasa perangkat lunak) Saya rasa itu saja dari saya. Senang bertemu dengan kalian semua. Terima kasih!
